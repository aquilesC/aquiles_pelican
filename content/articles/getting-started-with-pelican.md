# Getting Started with Pelican

Pelican is a static site generator written in Python. It's perfect for blogs, portfolios, and documentation sites.

## Installation

```bash
pip install pelican markdown
```

## Basic Setup

1. Create a new directory for your site
2. Initialize Pelican with `pelican-quickstart`
3. Create content in the `content/` directory
4. Generate your site with `pelican content`

## Customization

Pelican is highly customizable through:
- Themes
- Plugins
- Configuration files
- Custom templates

## Conclusion

Pelican is a powerful tool for building static websites. With its Python-based workflow and extensive customization options, it's a great choice for developers.
