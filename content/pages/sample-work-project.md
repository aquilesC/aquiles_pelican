# Sample Work Project

This is a sample work project page that demonstrates how to use the new portfolio-style work page template.

## Overview

This project showcases the capabilities of the work page template, including:

- **Hero sections** with gradient backgrounds
- **Key metrics** and achievements
- **Project highlights** in card format
- **Technology tags** and metadata
- **Call-to-action** sections

## Project Details

The work page template provides a professional, portfolio-style presentation that's distinct from regular articles. It uses a teal color palette instead of the standard blue, creating a visual distinction for work-related content.

### Key Features

1. **Visual Hierarchy**: The template uses larger headings and more spacing to create a premium feel
2. **Metrics Display**: Showcase key achievements with prominent numbers
3. **Project Highlights**: Feature specific accomplishments in card format
4. **Technology Stack**: Display technologies used in the project
5. **Professional CTAs**: Clear calls to action for collaboration and contact

## Technical Implementation

The template supports various metadata fields:

- `Template: work-page` - Uses the work page template
- `Category` or `work_type` - Categorizes the work
- `Company` - Company name
- `Role` - Your role in the project
- `Technologies` - Comma-separated list of technologies
- `Hero_image` - Main image for the project
- `Metrics` - Pipe-separated metrics (format: `value:label|value:label`)
- `Highlights` - Pipe-separated highlights (format: `title::description::url`)
- `Project_url` - Link to the live project

## Results

This template makes it easy to showcase your professional work in a visually appealing and organized manner, helping visitors understand your achievements and expertise at a glance.
